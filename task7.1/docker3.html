COMMANDS AND DESCRIPTION

systemctl start docker To start Docker Services
systemctl enable docker To make Docker Services permanent
docker images To see available Docker Images
docker ps To check the running containers
docker pull {image-name}:{tag-name} To pull your desired Docker Image
docker run -dit --name {give-name} {image-name}:{tag-name} To launch a Docker Container
docker rm -f {container-name} To remove a container
docker start {container-name} To start a running container
docker stop {container-name} To stop a running container
docker exec {container-name} {command-name} To exceute cmd inside docker container
